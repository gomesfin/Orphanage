# AfricaHomeHaven.org Website

Welcome to the official GitHub repository for the AfricaHomeHaven.org website. This repository contains the source code and assets for our platform, dedicated to empowering communities and changing lives across Africa.

## üìñ About The Project

AfricaHomeHaven.org is a non-profit organization committed to providing sustainable solutions for housing, education, and healthcare for orphaned and vulnerable children in East Africa. Our website serves as a central hub for our community, supporters, and partners to learn about our mission, get involved in our projects, and see the impact of their contributions.

This project is open-source, and we welcome contributions from developers, designers, and content creators who are passionate about our cause.

## üõ†Ô∏è Technologies & Frameworks

This website is built using a combination of standard web technologies and popular open-source libraries to create a rich, interactive experience.

* **HTML5 & CSS3:** For the core structure and styling.
* **Bootstrap:** As the primary CSS framework for layout and responsive design.
* **JavaScript (ES6):** For client-side interactivity.
* **jQuery:** As the main JavaScript library for DOM manipulation and event handling.
* **Google Fonts:** Using the "Overpass" and "Dosis" font families.
* **Fathom Analytics:** For privacy-focused website analytics.

### JavaScript Libraries

* **Owl Carousel:** For creating responsive and touch-enabled carousels.
* **Animate.css & AOS:** For CSS animations on page load and scroll.
* **Magnific Popup & FancyBox:** For creating elegant image and content lightboxes.
* **jQuery Waypoints & Stellar.js:** For scroll-based events and parallax backgrounds.

### Iconography

* **Open Iconic, Ionicons, Flaticon, IcoMoon:** A comprehensive set of icon libraries for visual elements.

## üöÄ Getting Started

This is a static website and does not require any build steps or complex dependencies. To get a local copy up and running, follow these simple steps.

### Prerequisites

You'll need a modern web browser and a code editor to view and modify the files.

### Installation

1.  **Clone the repo**
    ```sh
    git clone [https://github.com/gomesfin/Orphanage](https://github.com/gomesfin/Orphanage)
    ```
2.  **Open the `index.html` file**
    Navigate to the project directory and open the `index.html` file in your web browser to view the site.
